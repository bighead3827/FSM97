# FSM97 足球经理复刻项目规格说明

## 1. 项目概述

复刻经典足球经理游戏FSM97，使用Unity引擎和Unity MCP工具链开发。玩家将扮演俱乐部经理角色，全面掌控球队运营，包括阵容管理、战术制定、转会运作、训练安排和俱乐部经营。

## 2. 核心功能模块

### 2.1 球队阵容管理
- 球员数据库（包含欧洲五大联赛球队及球员）
- 首发阵容调整
- 替补名单管理
- 球员能力值展示（速度、射门、防守、传球、头球、控球等）
- 球员位置分配（前锋、中场、后卫、门将）

### 2.2 战术策略系统
- 预设战术模板（433进攻、442防守、352等经典阵型）
- 自定义阵型布局
- 球员跑位指令
- 比赛中实时战术调整

### 2.3 转会市场
- 球探系统（挖掘潜力新星）
- 球员转会谈判（买入/卖出）
- 转会资金管理
- 球员合同管理

### 2.4 训练系统
- 18项专项训练项目
- 针对不同位置的训练方案
- 球员属性成长系统
- 训练疲劳度管理

### 2.5 俱乐部经营
- 球场管理
- 门票定价
- 赞助商合作
- 财政管理

### 2.6 比赛模拟
- 2D俯视角比赛画面
- 实时音效（球迷呐喊、裁判哨声）
- 比赛统计数据
- 赛后分析

## 3. 技术架构

### 3.1 前端技术栈
- Unity引擎（2D/3D混合）
- uGUI用户界面
- TextMeshPro文本渲染

### 3.2 数据结构
- ScriptableObject用于数据存储
- JSON数据持久化
- 玩家存档系统

### 3.3 核心系统
- GameManager（游戏主控制器）
- MatchEngine（比赛模拟引擎）
- UIManager（界面管理）
- DataManager（数据管理）
- SaveSystem（存档系统）

## 4. 用户界面设计

### 4.1 主菜单界面
- 新游戏
- 继续游戏
- 设置
- 退出

### 4.2 主游戏界面
- 顶部信息栏（日期、资金、积分榜）
- 左侧导航菜单（阵容、战术、转会、训练、经营、比赛）
- 中央内容区域
- 底部状态栏

### 4.3 阵容管理界面
- 球员列表
- 阵型图
- 球员详细信息面板

## 5. 数据结构设计

### 5.1 球员数据结构
```
Player:
  - id: int
  - name: string
  - age: int
  - position: string
  - attributes:
    - speed: int (0-99)
    - shooting: int (0-99)
    - passing: int (0-99)
    - tackling: int (0-99)
    - heading: int (0-99)
    - stamina: int (0-99)
    - morale: int (0-99)
  - value: float
  - wage: float
  - contractYears: int
  - currentTeam: string
```

### 5.2 球队数据结构
```
Team:
  - id: int
  - name: string
  - league: string
  - players: List&lt;Player&gt;
  - tactics: Tactics
  - finances: Finances
  - stadium: Stadium
```
